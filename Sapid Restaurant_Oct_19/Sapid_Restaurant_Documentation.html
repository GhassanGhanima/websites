Sapid Website Introduction

Sapid - Restaurant HTML Template is a Modern, Clean and Professional site template. This Template is best suited for Restaurants, Cafes, Pub, Bars, Coffee House, Cafeteria, and any food business that needs an attractive and effective way to promote your food-related services. The layout looks beautiful on any Responsive Device, be it a laptop screen, iPad, iPhone, Android Mobile, or tablet.

Template Features

  • Easy to change colour (scss variables)
  • Modern, Unique Design
  • Beautiful animations and effects
  • Based on Bootstrap5
  • Smooth page transitions
  • Without page refreshing
  • PSD mockups included
  • Super fast loading
  • Valid HTML5 & CSS3
  • Responsive 100%
  • Easy Customization
  • Friendly & Quick Support

Sass Installation Guides

Step 1- Install Node Js

The first need is that you have to install Node js from https://nodejs.org/en/download/ on your PC.

Step 2- Install Ruby

The second need is you have to install Ruby from https://rubyinstaller.org/.

Step 3-Put Command line in Command Prompt

npm install -g sass

Step 4-For Start Sass

watch scss:css

Go to top.
